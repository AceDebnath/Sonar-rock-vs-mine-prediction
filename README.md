# Sonar-rock-vs-mine-prediction
A Machine Learning project that predicts whether an object is a rock or a mine using Linear Regression. The model is trained on the Sonar dataset with data preprocessing, training, and evaluation to achieve accurate binary classification results. Built with Python, Pandas, NumPy, and Scikit-learn.
